gauntlet
GitHub Docs PyPI

Prompt injection detection for LLM applications.

Runs locally. Bring your own keys.

Terminal 
$ pip install gauntlet-ai[all]

How it works

LAYER 1

Rules

50+ regex patterns. Zero dependencies. Catches ~60% of attacks.

FREE
LAYER 2

Embeddings

Semantic similarity against 500+ known attack vectors.

REQUIRES OPENAI KEY
LAYER 3

LLM Judge

Claude Haiku analyzes text characteristics for sophisticated attacks.

REQUIRES ANTHROPIC KEY

Code example

Python API 
from gauntlet import Gauntlet

g = Gauntlet()
result = g.detect("ignore previous instructions")
print(result.is_injection)  # True
print(result.confidence)    # 0.95
CLI 
$ gauntlet detect "ignore previous instructions"
  INJECTION DETECTED
  Layer 1 | Confidence: 95% | Type: instruction_override

$ gauntlet scan ./prompts/ --all
  Scanned 42 files: 3 flagged, 39 clean

MCP Integration

Use gauntlet with Claude Desktop or any Model Context Protocol client. Add this to your MCP configuration:

claude_desktop_config.json 
{
  "mcpServers": {
    "gauntlet": {
      "command": "gauntlet",
      "args": ["mcp-serve"]
    }
  }
}
